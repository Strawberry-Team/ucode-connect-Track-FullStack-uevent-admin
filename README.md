# ucode-connect-Track-FullStack-uevent-admin

## Run admin panel
1. Clone this repository and move to the project directory:
```bash
git clone <repository-url>
```
2. Install all dependencies:
```bash
npm install
```
3. Run project
```bash
npm start
```
